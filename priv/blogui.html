Distel for Erlang Mode

I installed Distel today. Distel is a fantastic bit of software that basically turns Emacs into an Erlang node whenever erlang-mode is invoked. An Erlang node that can communicate using Elisp. Many thanks to Luke Gorrie and co for serious greatness.
